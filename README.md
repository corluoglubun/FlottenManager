# FlottenManager 🚗

Ein Java-Konsolenprogramm zur Verwaltung von Firmenfahrzeugen und Buchungen.

## Technologien
- Java 11
- MariaDB
- JDBC

## Funktionen
- Fahrzeuge verwalten
- Mitarbeiter verwalten  
- Fahrzeuge buchen und stornieren
- Datenpersistenz über Datenbank
- Business Rules (max. 2 Buchungen pro Mitarbeiter)

## Projektstruktur
- `Fahrzeug.java` — Klasse für Fahrzeuge
- `Mitarbeiter.java` — Klasse für Mitarbeiter
- `Buchung.java` — Klasse für Buchungen
- `Flottenverwaltung.java` — Verwaltungslogik
- `DatenbankManager.java` — Datenbankverbindung
- `Main.java` — Hauptprogramm mit Menü
